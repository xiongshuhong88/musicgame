# 🕺 华语金曲歌舞厅 (Retro Music Hall)

**《华语金曲歌舞厅》** 是一款融合怀旧美学与律动玩法的动作音乐游戏。

🎮 **游戏简介**
玩家将扮演一名像素风复古小人，置身于上世纪90年代的霓虹歌舞厅，穿梭于《迷迭香》、《龙卷风》等经典金曲的专属房间中。游戏采用独特的**“节奏接宝”**玩法：伴随着沉浸式MV背景，你需要在地板上灵活左右横跳，接住随节奏下落的音符与星星。

🍄 **特色机制**
游戏致敬了经典红白机元素，加入了**马里奥式变身系统**：
- **变身蘑菇**：吃到后体型巨大化，判定范围翻倍，轻松接宝！
- **无敌星星**：吃到后全身闪烁无敌光效，分数双倍奖励！

在复古CRT滤镜与动感节拍中，重温那段流金岁月，挑战手速与节奏的极限吧！

---

## 🛠️ 如何运行

1. 克隆项目
\`\`\`bash
git clone https://github.com/xiongshuhong88/musicgame.git
\`\`\`

2. 安装依赖
\`\`\`bash
npm install
\`\`\`

3. 启动游戏
\`\`\`bash
npm run dev
\`\`\`
